== Description ==

This YouTube plugin allows you to show your recently listened uploaded YouTube videos, and display them in a list (With links) as a widget on your blog.

You can also display videos from search results - Simply enter the search query and then a list of videos will be shown, related to the query you entered!

You just need to insert your YouTube username and the number of videos you want in the list, and everything will be managed for you. You just need to enable the widget.

== Installation ==

1. Transfer the file `the-youtube-plugin.php` to the `/wp-content/plugins/` folder
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Edit the YouTube Plugin Settings, through the Admin Panel --> Settings panel.
4. Enable the Widget through the Appearance --> Widgets tab.
