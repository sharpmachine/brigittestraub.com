== Description ==

You can play videos from youtube or flv files. Create your own selection of videos and share on your wordpress. Very easy to use and with a spectacular design with just a few seconds of installation and configuration. It is specially designed for use as widget.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload the `easy-video-widget` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Drag&Drop the widget in your sidebar
4. write the urls of the videos, one url by line. 
5. You can also configure the size of the widget, the number of thumbnails, titles and the autoplay option.
